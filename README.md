# denos

This is a collection of typescript code developed with `Deno` as the runtime, published to `jsr`.

## Packages

The following is a complete list of packages in the repository:

- [matrices](#matrices)
- [scripts](#scripts)

### matrices

A library containg a `Matrix` class with associated matrix operations.

### scripts

A collection of different Deno scripts – these are primarily used for CI-CD pipelines where the deno runtime is available to the runner.
